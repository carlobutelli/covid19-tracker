# Covid19 tracker
-----------------
This Web App shows the total number of Coronavirus cases registered today around the world.

---

## Start the App locally
```bash
mvn spring-boot:run
```

---

## Build API
-----------

## Start docker container
```bash
docker-compose up -d
```
## Package the API in a Jar
```bash
mvn package
```

---

## Run API
---------
```bash
java -jar target/covid19-tracker.jar
```
